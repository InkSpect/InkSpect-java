# InkSpect-java

InkSpect for Java

## Prerequisite

Install development tools:
- sbt
- java 

Or enable Flake for Nix environment:

```bash
direnv allow
```

## Build

```
sbt clean stage
```

## Run

- Run CLI (Command Line Interface)

```
./inkspect_java_cli
```

- Run REPL (Read-Eval-Print-Loop) for interactive usage

```
./inkspect_java_repl
```
